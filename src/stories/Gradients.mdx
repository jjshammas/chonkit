<Title>Gradients</Title>
<Subtitle>Pixel dithering and directional gradient patterns with Box</Subtitle>

Pixel art often uses dithering to create gradients and shading. Unlike standard CSS gradients,
Box gradients are intentionally pixelated and step between colors to keep a retro visual style.

Gradients are defined as strings, similar to CSS gradients:

```javascript
90deg, #ff0000 20%, pink, primary 80%
```
